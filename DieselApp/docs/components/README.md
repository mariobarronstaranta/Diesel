## Convenciones usadas

- **Dependencias principales**: librerías externas y componentes internos.
- **Props**: propiedades esperadas por el componente.
- **Estado interno**: `useState` y control local.
- **Integración de datos**: tablas/rpc de Supabase, cuando aplica.
- **Comportamiento**: validaciones, flujo de usuario y reglas de negocio implementadas.
